Day 20 - While Loops

Key Concepts
- `while` loop - checks condition before executing
- `do-while` loop - executes at least once, then checks condition
- Loop validation (input validation)
- Infinite loops and how to avoid them

- While loops are ideal when you don't know the number of iterations beforehand
- Always ensure the loop condition will eventually become false
- do-while is useful when you need to run code at least once
