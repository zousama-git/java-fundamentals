Day 20 - Arrays & Looping

- Arrays store multiple values of the same type
- Array elements are accessed using an index (starting at 0)
- You can modify array values using their index
- array.length gives the number of elements
- Traditional for loops allow index-based iteration
- Enhanced for loop (for-each) simplifies iteration
- Arrays can be sorted using Arrays.sort()
- Arrays can be filled with a single value using Arrays.fill()
